# MailSender

# Pre-requisites
- install [Nodejs](https://nodejs.org/en/blog/release/v18.16.1) version v18.16.1

# Getting Started
- clone the repository

  ` git clone <https://github.com/HilaryEmmanuel/Mailer.git> `

- Install dependencies
  
  ` cd <myapp> `
  ` npm install `

- Build and run the project
  
  ` node node ./bin/www `

# Technology used
- Nodejs
- Express js
- Nodemailer
